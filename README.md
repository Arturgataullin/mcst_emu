**Описание структуры проекта**
-

**assembler** - переводит текст программы в машинный код\
**emulator** - исполняет машинный код\
**common** - содержит общие типы, утилиты\
**examples** - примеры готовых команд на реализуемом ассемблере\
**test** - тесты для каждого модуля проекта и полного пайплайна\
**docs** - описание процессора\

**Описание сборки проекта**
-
mkdir build в корневой папке проекта\
cd build\
cmake ..\
make\
build/assembler/assembler - исполняемый файл для ассемблера\
build/emulator/emulator - исполняемый файл для эмулятора\
build/tests - тесты\
