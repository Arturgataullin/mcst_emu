**Описание процессора**
=
**Регистры**
-
Процессор имеет 16 целочисленных 32-разрядных регистров общего назначения.  
Регистры обозначаются Rn (R0, R1.. R15).  
В начале работы регистры инициализируются нулем  
**Инструкции**
-
Размер инструкции - 4 байта.  
**Формат описания инструкций:**  
Общее описание на естественном языке  
Алгоритм работы на C-подобном языке  
Текстовое представление для составления программы  
Двоичное представление для исполнения процессором в виде последовательности байт  
**Описание инструкций:**  
Загрузка литерала в регистр (Load Immediate)  
 R[a] = I = I0 | (I1 << 8)  
 LI Ra I  
 0x00 a I0 I1  
Сложение двух регистров  
 R[a] = R[b] + R[c]  
 ADD Ra Rb Rc  
 0x02 a b c
